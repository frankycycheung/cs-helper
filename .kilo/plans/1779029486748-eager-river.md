# CS Upgrade News & Workflow Management Panel Implementation Plan

## Overview
Build a modern web interface for CS representatives to generate "Upgrade News" (New Enhancements section) and manage pre-deployment QA workflow using Vue 3 (Composition API) and Bootstrap 5.

## Design Requirements
- Three-column dashboard layout on desktop (stacked on mobile)
- Clean SaaS aesthetic with Bootstrap 5 utilities
- Color palette: Primary Blue, Success Green, Warning Amber
- Real-time preview with validation modal

## Component Structure

### 0. Sidebar Component (`src/components/Sidebar.vue`)
- Left sidebar with toggle functionality INSIDE the sidebar
- Accepts sidebar state as prop or manages its own state
- Shows full route titles with SVG icons when open
- Shows only SVG icons when closed (compact mode)
- Contains navigation links to different pages (Home, Upgrade News)
- Uses Vue router-link for navigation
- Toggle button positioned at top of sidebar

### 1. Header Component (`src/components/common/Header.vue`)
- Sticky navbar with project title, status badge, and user placeholder
- Bootstrap classes: `navbar navbar-expand-lg navbar-light bg-white border-bottom`
- Note: Sidebar toggle functionality moved to Sidebar component per user request
- Note: Sidebar toggle functionality moved to Sidebar component per user request

### 2. Main Dashboard (`src/components/upgrade-news/UpgradeNewsDashboard.vue`)
- Container with three-column layout using Bootstrap grid
- Left column: Input & Configuration
- Center column: QA Workflow Checklist  
- Right column: Live Preview & Export

### 3. Input & Configuration Panel (`src/components/upgrade-news/InputConfigPanel.vue`)
- Card with title "1. Module & Content Configuration"
- Module selector (searchable dropdown)
- Feature title input
- Raw description textarea
- AI Polish button
- Usage paths (Web/App)
- Internal remarks textarea (distinct styling)

### 4. QA Checklist Panel (`src/components/upgrade-news/QAChecklistPanel.vue`)
- Card with title "2. Pre-Publish Quality Checklist"
- Three checkbox groups using Bootstrap list groups:
  - Environment & Scope
  - Function & Flow  
  - Records Alignment
- Each checkbox as large clickable row with hover state

### 5. Live Preview Panel (`src/components/upgrade-news/LivePreviewPanel.vue`)
- Card with title "3. Upgrade News Live Preview"
- Rendered preview area mimicking official announcement
- Dynamic content binding from input fields
- Generate/Copy News button (primary action)

### 6. Warning Modal (`src/components/common/WarningModal.vue`)
- Bootstrap modal for incomplete checklist warnings
- Dynamic listing of unchecked items
- Go Back and Proceed Anyway buttons

## State Management Plan
- Use Vue `ref` for all form inputs
- Reactive array for checklist states
- Computed properties for live preview generation
- Internal remarks excluded from preview logic
- Modal control state for warning dialogs

## Implementation Approach
1. Create reusable components in `/components/common`
2. Create upgrade-news specific components in `/components/upgrade-news`
3. Implement state binding with v-model
4. Add real-time preview updates
5. Implement warning modal logic
6. Add clipboard copy functionality
7. Ensure responsive behavior (desktop 3-column, mobile stacked)
